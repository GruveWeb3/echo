Gruve Events Widget - Plain JS Embed

Manual Initialization

You can also manually initialize the widget using JavaScript:

// Create a button element
const button = document.createElement('button');
button.textContent = 'Custom Button';
button.id = 'my-custom-button';
document.getElementById('manual-init-container').appendChild(button);

// Initialize Gruve CTA on the button
new GruveCTA({
  targetElement: button,
  eventAddress: '0xf5B129B9abC19D591F801Ad8Edc7Ae747E031adB',
  isTest: true,
  config: {
    themeColor: '#27ae60',
    displayText: 'Custom Button'
  }
});
